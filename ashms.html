Additive Subtractive Hybrid Machining System
Add material only where needed, remove for precision.
Project Type: Academic Project

This project started of as a personal project, but later was used as a platform for Hybrid Machining System.


Designed to take advantages of both additive and subtractive manufacturing.

It is best suited for making molds for vacuum forming, large size bulky parts with use of Pellet Extruder it prints at higher material deposition rate than conventional FFF 3D Printers.

Secondary tool head with tool changing system is only for lightweight operations such as 3D Printing and Laser Engraving etc.

It uses "hexrail" linear motion system that I developed on hexagonal bars and 120degree v groove double race bearings. which provides very good rigidity over other systems like open builds delrin v wheels.
